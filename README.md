Introduction:

In this project, we will analyze Zomato’s restaurant dataset using Python with Pandas library to find meaningful insights.
We aim to answer questions such as:

1. Do more restaurants provide online delivery compared to offline services?
2. Which types of restaurants are most favored by the general public?
3. What price range do couples prefer for dining out?

---------------------------------------------------------------------------

Python Libraries Used:

Here, we used python libraries built for Data analysis & visualization such as Pandas, Seaborn, Matplotlib etc.

To install Pandas, run command: pip install pandas
To install Seaborn, run command: pip install seaborn
To install Matplotlib, run command: pip install matplotlib

-------------------------------------------------------------------------------------------------

Source data:

Source file used here is Zomato-data.csv, which contains information like restaurant names, types, ratings, votes, online order availability etc..
Source file is downloaded from this link, https://www.geeksforgeeks.org/data-analysis/zomato-data-analysis-using-python/

-----------------------------------------------------------------------------------------------------------------

Our Findings:

1. Most popular restaurant category is Dining restaurants.
2. The most voted restaurant is Empire Restaurant.
3. Majority of restaurants do not accept online orders.
4. Typically, restaurants receive rating ranging from 3.8 to 4.2.
5. Majority of couples prefer restaurant with approx. cost of 300 rupees.
6. Restaurants accepting online orders received much higher ratings compared to the restaurants that do not accept online orders.
7. Majority of Dining restaurants do not accept online orders while majority of Cafes accept online orders.
